# GitHub Actions Workflow Examples

This repository is dedicated to learning and experimenting with **GitHub Actions** by writing workflows in **YAML**.  
The goal is to understand how automation works in GitHub and how we can use workflows for CI/CD, testing, and deployment.

GitHub Actions is a CI/CD (Continuous Integration and Continuous Deployment) platform that allows you to:
- Automate tasks directly in your GitHub repository.
- Run workflows triggered by events (like `push`, `pull_request`, or scheduled jobs).
- Build, test, and deploy applications seamlessly.
